# ScanCardNum
**一个基于CardIO SDK，自定义UI的银行卡识别Demo。**

**【注意：】由于CardIO SDK过大，上传项目的时候我把它remove掉了。**
**使用时请打开PodFile里面第11行：`pod 'CardIO', '~> 5.4.1'`的注释**
**并cd到工程目录下执行'pod install'命令，加载CardIO库**
